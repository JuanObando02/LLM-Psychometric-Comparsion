1.   Introducción

Sección

1.1    Propósito

Información

[NEEDS-14] El objetivo principal de este documento es describir las necesidades de los usuarios del aplicativo web por medio de historias de usuario al momento de comparar sus rasgos no cognitivos.

Sección

1.2    Alcance

Información

[NEEDS-15] Este documento especifica las necesidades de los usuarios que accederán al aplicativo web que permite evaluar y comparar rasgos no cognitivos entre modelos LLM y grupos específicos.

Información

[NEEDS-16] Los usuarios deben poder:

· Realizar inventarios psicométricos estandarizados.

· Visualizar resultados de grupos poblacionales y LLM. 

· Comparar resultados entre LLM y grupo poblacional.

· Descargar resultados de inventario psicométrico.

 

 

​

Sección

1.3    Roles de Usuario

Sección

1.3.1    Usuario

Información

[NEEDS-12] El usuario es la persona que ingresa al aplicativo web para realizar las pruebas pscicométricas, visualizar sus resultados y su comparación con el LLM.

Sección

1.3.2    Investigador

Información

[NEEDS-13] El Investigador es una persona que ingresa al aplicativo web para realizar un estudio de rasgos psicométricos no cognitivos entre un grupo poblacional y distintos LLM, para identificar similitudes y diferencias en sus rasgos.

Sección

2.   Historias de Usuario

Sección

2.1    Sección Informativa

Historia de usuario

2.1.1    Información General

[NEEDS-19]

Usuario

Como usuario quiero conocer qué tipo de pruebas realizaré para comprender mejor el contenido 

El aplicativo muestra una sección informativa sobre las pruebas psicométricas.

Sección

2.2    Evaluación de rasgos

Historia de usuario

2.2.1    Aplicación de Inventario psicométrico

[NEEDS-20]

Investigador

Quiero realizar las pruebas psicométricas para evaluar mis rasgos no cognitivos.

·  El aplicativo muestra las preguntas de los inventarios psicométricos.

·  El usuario puede responder las preguntas.

·  El aplicativo calcula los resultados de la prueba realizada.

Historia de usuario

2.2.2    Visualización de resultados

[NEEDS-21]

Usuario

Quiero ver mi reporte visual con la especificación de mis resultados por rasgo.

·  El aplicativo genera un reporte visual.

·  Se visualiza una gráfica que contiene el nivel cada rasgo no cognitivo.

Historia de usuario

2.2.3    Reporte de Resultados
[NEEDS-23]

Usuario

Quiero descargar el reporte de los resultados de mis rasgos no cognitivos para tener un registro propio de estos. 

·  El aplicativo muestra el reporte de los resultados de la prueba.

·  El aplicativo permite descargar el reporte en un archivo PDF.

Historia de usuario

[NEEDS-24]

Usuario

Quiero descargar el reporte comparativo de los resultados de mis rasgos no cognitivos frente al perfil de los LLM. 

·  El aplicativo muestra el reporte comparativo entre los resultados del usuario y los LLM.

·  El aplicativo permite descargar el reporte comparativo en un archivo PDF.

Historia de usuario

2.2.4    Plantilla Prueba Psicométrica

[NEEDS-25]

Investigador

Requiero descargar una plantilla de la prueba psicométrico para aplicarla a un grupo poblacional.

· El aplicativo permite descargar la plantilla de la prueba psicométrica en un archivo de Excel.

Historia de usuario

2.2.5    Carga de respuestas

[NEEDS-26]

Investigador

Quiero cargar el archivo de los resultados de la prueba psicométrica aplicada.

· El archivo cumple con el formato de carga.

· El aplicativo calcula los resultados del archivo cargado.

Historia de usuario

2.2.6    Resultados Obtenidos
[NEEDS-27]

Investigador

Quiero ver un reporte visual comparativo entre el grupo poblacional frente a los LLM, para identificar similitudes y diferencias en sus perfiles no cognitivos.

·  Se muestra la gráfica comparativa que contiene los niveles de cada rasgo no cognitivo entre el grupo poblacional y los LLM.

Historia de usuario

[NEEDS-28]

Investigador

Quiero descargar el reporte de los resultados del grupo poblacional para obtener un registro comparativo frente a los LLM.

·  El aplicativo permite descargar un archivo PDF de resultados.

Sección

3.   Restricciones

Restricción

3.1    Privacidad

[NEEDS-29]

Usuario

No quiero que se almacenen datos personales ni mis resultados de las pruebas.

El aplicativo no tendrá base de datos asociada.

Restricción

3.2    Compatibilidad

[NEEDS-31]

Usuario

Quiero que el aplicativo se encuentre disponible desde cualquier navegador y cualquier dispositivo.

El aplicativo se visualiza desde chrome, firefox, o safari y dispostivios moviles, de escritorio o tablets.

Sección

3.3    Tamaño de Documento

Restricción

3.3.1    Reporte de Resultados

[NEEDS-34]

 

Usuario

Quiero que la descarga de mis resultados no supere las 5MB de tamaño 

El aplicativo genera archivo de reporte PDF por debajo de las 5MB de tamaño.

Restricción

3.3.2    Resultados Obtenidos

[NEEDS-35]

Investigador

Quiero que la descarga del reporte de resultados comparativos no supere las 5MB de tamaño 

El aplicativo genera archivo de reporte PDF por debajo de las 5MB de tamaño.

Restricción

3.4    Tamaño de Carga Plantilla

[NEEDS-32]

Investigador

Quiero que el aplicativo evalúe las respuestas de al menos 50 personas. 

El aplicativo calcula los rasgos no cognitivos de más de 50 personas.

Restricción

3.5    Restricciones de Vistas

[NEEDS-33]

Usuario

Quiero hacer uso del aplicativo web sin interrupciones.

El aplicativo soporta visitas simúltaneas de al menos 200 personas.
